# How to Git

1# Create a repository.

2# Fork the repository.

3# Clone the repository.
```c
git clone <repository>
```
4# Create a branch
```c
git checkout -b <branch-name>
```

5# Work on files.

6# Add files
```c
git add <filename>
```

7# Comment files
```c
git commit -m "<commentary>"
```
8# Pushing code
```c
git push origin <branch-name>
```
